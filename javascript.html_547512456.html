Wie aktiviere ich Javascript in meinem Browser?

Du musst Javascript in deinem Browser aktivieren, um alle Features der Community nutzen zu können.

In dem Feld rechts kannst du testen, ob Javascript erfolgreich aktiviert wurde. Nur wenn das Feld grün ist, wurde dein Browser richtig eingestellt.

Falls Javascript bei dir noch nicht aktiviert ist, folge bitte diesen Anweisungen:

Internet Explorer (6.0)

  1. Wähle aus dem Menü in der oberen Leiste die Option "Extras".
  2. Wähle den Menüeintrag "Internetoptionen".
  3. Klicke auf die Registerkarte "Sicherheit".
  4. Klicke auf "Stufe anpassen".
  5. Scrolle nach unten bis zu dem Abschnitt mit der Überschrift "Scripting".
  6. Wähle unter "Active Scripting" die Option "Aktivieren" aus und klicke auf "OK".

Netscape Navigator (4.8)

  1. Wähle aus dem Menü in der oberen Leiste die Option "Bearbeiten".
  2. Wähle "Einstellungen".
  3. Wähle die Option "Erweitert".
  4. Wähle "Skripts & Plugins"
  5. Aktiviere das Kontrollkästchen "JavaScript aktivieren" und klicke auf "OK".

Mozilla Firefox (1.0)

  1. Wähle aus dem Menü in der oberen Leiste die Option "Extras".
  2. Öffne den Menüpunkt "Einstellungen".
  3. Wähle auf der linken Seite die Option "Web-Features".
  4. Aktiviere das Kontrollkästchen neben "JavaScript aktivieren" und klicke auf "OK".

Mozilla Firefox (1.5)

  1. Wähle aus dem Menü in der oberen Leiste die Option "Extras".
  2. Öffne den Menüpunkt "Einstellungen".
  3. Wähle aus dem Menü in der oberen Leiste die Option "Content" aus.
  4. Aktiviere das Kontrollkästchen neben "JavaScript aktivieren" und klicke auf "OK".

Apple Safari (1.0)

  1. Wähle aus dem Menü in der oberen Leiste die Option "Safari".
  2. Wähle "Einstellungen".
  3. Wähle die Option "Sicherheit".
  4. Aktiviere das Kontrollkästchen neben "JavaScript aktivieren".

Um zu testen ob die Einstellungen erfolgreich übernommen wurden, klicke bitte hier: Javascript testen.
